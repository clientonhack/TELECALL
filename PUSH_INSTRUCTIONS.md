# Инструкции по заливке кода на GitHub

Для заливки исправленного кода на GitHub выполните следующие шаги:

## 1. Если у вас есть существующий репозиторий на GitHub:
```bash
git remote add origin <URL_вашего_репозитория_на_GitHub>
git branch -M main
git push -u origin main
```

## 2. Если вы создаете новый репозиторий на GitHub:
1. Создайте новый репозиторий на GitHub.com
2. Скопируйте URL репозитория
3. Выполните команды:
```bash
git init
git add .
git commit -m "Исправлены ошибки в lambda функциях в python.py"
git remote add origin <URL_вашего_нового_репозитория>
git branch -M main
git push -u origin main
```

## 3. Актуальное состояние репозитория:
На данный момент в репозитории уже содержится исправленный файл python.py с фиксом ошибок в lambda-функциях, которые выявлены в задании.
